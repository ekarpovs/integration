### TH-UI-PROXY ###
Provides the UI pack of templates for the th-client(s)

#### Config ####
Edit the .env file


#### Run ####
```bash
./start.bash
```

#### Stop ####
```bash
.stop.bash [down]
```

#### Connect to the docker container ####
```bash
http://127.0.0.1:8022
```

#### Test via Swagger ####
```bash
http://127.0.0.1:8022/docs
```

#### Connect inside the docker container ####
```bash
docker exec -it th-ui-proxy-evgeny bash
```
