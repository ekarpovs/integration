### SamApp ###

#### Config ####
Edit the .env file

#### Run ####
```bash
./start.bash
```

#### Stop ####
```bash
.stop.bash [down]
```

#### Connect to the docker container ####
```bash
http://127.0.0.1:8051
```

#### Test via Swagger ####
```bash
http://127.0.0.1:8051/docs
```

#### Connect inside the docker container ####
```bash
docker exec -it sam-app-evgeny bash
```
